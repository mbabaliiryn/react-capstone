##  Catelogue of Recipes
This is an application that fetches data from the URL and diplays it on the homepege
A user has the ability to filter a group of the commodities by their company name for example. A user has the ability to view each commodity in details by clicking a details button provided below each commodity. Once a user is in the path of a single commodity, he or she can go back by clicking the home button provided below the description of the commodity

- Redux

- Bootstrap
